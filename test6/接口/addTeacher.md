# 接口：addTeacher  [返回](../README.md)
用例： [录入教师信息](../用例/录入教师信息.md)
- 功能：
    添加教师。
    
- 权限：
    管理员。    
    
- API请求地址： 
    接口基本地址/v1/api/addTeacher

- 请求方式 ：
    POST

- 请求实例：

        {
            "teacher_id":533813
            "name":"zwd",
            "department_id":"计算机"
        }
        
- 请求参数说明:        

  |参数名称|说明|
  |:---------:|:--------------------------------------------------------|      
  |teacher_id|教师工号|
  |name|教师姓名| 
  |departemnt_id|所属院系id|
  
- 返回实例：

        { 
            "status": true,
            "info": null,    
        }
 
- 返回参数说明：    
 
  |参数名称|说明|
  |:---------:|:--------------------------------------------------------|      
  |status|bool类型，true表示正确的返回，false表示有错误|
  |info|返回结果说明信息|
